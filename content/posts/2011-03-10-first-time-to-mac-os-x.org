不过 Mac 平台的封闭性带来一个众人皆知的事实，那就是 Hacintosh 的成功性与人品  ------ 也无关，我的意思是说，Hacintosh的成功与否完全是莫名其妙的事情，无论是虚拟机还是实体机，能否 Hacintosh，要看上帝的心情。

于是转投 [[http://www.vmware.com/][VMware]]，我按照 VMware 的要求，下载了 300+M 的 VMware7.1的 bundle，按照  [[https://wiki.archlinux.org/index.php/VMware][ArchWiki]] 上的步骤，成功装上了 VMware，但是确始终启动不起来。无奈，转投 Windows 吧。之所以坚持 Linux，是因为我需要 LaTeX，我需要  Emacs+Auctex+cdlatex+outline-mode 来完成我的毕业论文啊。

好在 Windows 下的 VMware 还算正常，安装过程也算顺利，我就不在细说了。具体过程可以参考《[[http://blog.csdn.net/zhanglei5415/archive/2010/12/22/6091717.aspx][VMWare下安装 MAC
OS X Snow Leopard 10.6]]》，需要注意的主要是：

- 安装之前用  [[http://www.grc.com/securable.htm][securAble]] 检测下 CPU 是否支持虚拟化；
- 内存至少 1G，最好主机配上 4G 内存，而 VMware 配上 2G 内存，否则慢起来很要命，如果搞开发，硬盘最好 20G 以上；
- 注意修改 VMware 目录下的  .vmx 下的 guestOS 由 guestOS =
   "freebsd-64"改为 guestOS = "darwin10"；
- 启动时始终需要  darwin.iso 来引导；
- 成功安装系统后安装  darwin.iso 里面的 vmtools，获得更好地使用体验；
